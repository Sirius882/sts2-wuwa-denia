◆ 全新的可玩角色 达妮娅

◆ 现有80+新卡

◆ 使用两种特殊能量：
· 虚质 —— 通过攻击、卡牌等方式获取，在特定情形下强化卡牌效果
· 黯核 —— 通过粉色形态回合进程自然获取或通过特定卡牌获取，在特定情形下强化卡牌效果

◆ 形态切换机制
· 粉色形态：常态达妮娅，擅长运转和积累特殊能量
· 黑色形态：鸣式共鸣者达妮娅，擅长输出

◆ 全新通用系统：集谐

基础牌和普通牌最多把目标堆到1层集谐，罕见牌最多堆到2层，稀有牌最多堆到3层。进入共鸣模态·集谐后，基础、普通和罕见稀有度的牌各自能附加的最高集谐层数+1。

当附加有集谐的目标发生谐度破坏时，会在原本的谐度破坏结算完成后触发集谐结算。

集谐谐度破坏效果如下：
1层：获得1力量10聚爆轨迹。对被破坏的目标造成其最大生命值5%的伤害。
2层：获得2力量20聚爆轨迹。读取玩家抽牌堆、手牌和弃牌堆中的集谐响应牌。每张集谐响应牌按牌的类型产生一次效果：攻击牌对被破坏的目标造成3点伤害，技能牌使触发者获得3点格挡，其他类型的牌对被破坏的目标造成其最大生命值3%的伤害。
3层：获得3力量30聚爆轨迹。本回合玩家战斗中的所有牌免费；清除所有由临时效果附加的集谐响应；晕眩所有敌人；该目标在本回合内不能再被附加集谐。如果触发者是达妮娅，会把虚质和黯核补满。

最多通过集谐破坏获得6点力量和60层聚爆轨迹。达到上限后，后续集谐破坏仍会执行该层的其他效果，但不会再继续获得超过上限的力量或聚爆轨迹。
